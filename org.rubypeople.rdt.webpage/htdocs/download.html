Using the Eclipse Update Manager
  • Choose HelpSoftware UpdatesFind and Install
  • In the dialog choose "Search for new features to install" and push next
  • Select "Add Update Site", enter an arbitrary name and the url http://rubyeclipse.sf.net/updatesite
  • Check RDT and push next
  • Select the feature org.rubypeople.rdt and push next. RDT will be installed and be available after restart.

Nightly build

There is an automatic nightly build of the sources in CVS. Usually you do not want to use this. However, it might be useful if you know about certain features or bug fixes which have been done in CVS but are not yet part of the latest release. As common for this kind of build, there is no guarantee of the quality of this build. It might not even fully compile. A hint of the quality of this build gives the result of the (PDE) unit test suite. A last warning: the nighly build runs on the sourceforge compile farm. Sourceforge advises to inform end users about the possibility of unauthorized changes to the build result.
