Rename

Description

The Rename refactoring can handle several items in the Ruby code. Depending on the selection a different dialog is opened providing specific options for the refactoring process.
The following items can be renamed:
  • Class
  • Field
  • File
  • Local Variable
  • Method
  • Module

Conditions

  • To rename an item from the list above, the caret has to be set in its name. The refactoring recognizes itself what to rename.

    Options


    For the renaming refactoring there is usually only one option, the new name for the item.
    Altough some certain rename refactorings have a further selection page, where all depending call (i.e. a method call) can be checked for renaming too. Since Ruby is a dynamically typed language it is quite hard to find such calls automatically. So they have to be chosen by hand.
